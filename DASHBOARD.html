⚠ Layer 3 SQL Type Mismatch
RAG retrieval fails with varchar[] && text[] type mismatch. Layer 3 gracefully falls back to Layer 1+2 deterministic results.

Uncached parcel assessments take ~20s (ZIMAS dominates). Cached assessments: ~178ms. Mitigation: pre-cache parcels.
